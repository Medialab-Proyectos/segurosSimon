# Core Features

This document describes the main functional modules of the Simon application.

---

# Insurance Module

The insurance module allows users to manage and purchase vehicle insurance.

### Key features

- View existing policies
- Download policy documents
- Check policy status
- Renew policies

### Policy States

Policies can have the following statuses:

- Active
- Expiring soon
- Expired
- Renewed

---

# Insurance Purchase

Users can purchase two types of insurance:

### SOAT

Mandatory vehicle insurance in Colombia.

Users can:

- purchase online
- request advisor contact

---

### Auto Insurance

Optional vehicle coverage with different plan levels.

Plans may include:

- Basic
- Intermediate
- Full coverage

Users can:

- view coverage details
- request a quote
- purchase online
- request advisor contact

---

# Policy Document Access

Users can access policy documents directly from the app.

Features include:

- view policy details
- download PDF documents
- verify expiration dates

---

# Assistance Module (Future)

This module will provide mobility-related services.

Possible features:

- roadside assistance
- service requests
- emergency contacts

---

# Notifications and Reminders

The application should notify users about:

- policy expiration
- renewal opportunities
- relevant mobility alerts
